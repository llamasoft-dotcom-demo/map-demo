Instructions -

In helping clients understand their supply chains, maps are an important visualization tool. In this exercise we would like you to take some steps to help make this map more useful. This example uses Google Maps and their maps API to show information about airports. While we do not use Google Maps and the maps API, the types of visualizations and map interactions you are allowed with Google Maps are common. In order to do this exercise you will need to get an API key for Google Maps if you do not already have one. Put the API key in the appropriate place in this file. - DONE

Currently this page loads a list of airports into a drop down. You can select an airport and it"s code and city will display in the data table. A marker will be placed on the map displaying the airport. If you roll your mouse over the marker the airport"s code will be displayed.

The current code is very simple. Many improvements need to be made:

  • Fill in the rest of the data in the table when a user selects an airport. - DONE
  • Allow a user to remove a marker - DONE, button in InfoWindow, shown on marker click
  • Sort the list so it is easier to find airports - DONE

Some more things to think about

  • We are constantly struggling with displaying as much data as simply and streamlined as possible. Is there something you can do with the layout to show the same information using less screen real estate? - Use an InfoWindow instead, implemented.
  • Regarding markers, what happens with the existing code when you select an item multiple times? - Each time a location is selected, a new marker is created. Consquently, selecting the same location multiple times results in multiple, redundant markers on top of each other (cough cough bug ignored)
  • If a user selects an airport that is not currently shown on the map, how do they find it? Can you make this easier? - It now scrolls to that location automatically, as that is the "natural" expected behavior. Something popping up off screen is programmer level expected, which doesn't sell.
